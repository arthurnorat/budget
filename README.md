# Budget Tracker - iOS App

A simple and intuitive budget tracking application for iOS, built with UIKit. Track your expenses, monitor remaining budget, and visualize your spending habits.

## Features

- Add new expenses with name, amount, and type (fixed/variable)
- View summary of all expenses in a clean table format
- Visualize budget usage with progress indicators
- Responsive UI that works across different device sizes

## Technical Details

- **Architecture**: MVC pattern
- **UI**: Programmatic UI (no Storyboards)
- **Components**: UITableView, UIStackView, UIProgressView
- **Language**: Swift 5
